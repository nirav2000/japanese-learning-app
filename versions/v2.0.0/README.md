# Version 2.0.0 - "Study Mode & Auto-Rating Edition"

**Release Date:** February 5, 2026

## Major Features

### Study Mode
- Review mode checkbox for learning with answers visible
- Perfect for initial learning before self-testing
- Hides rating buttons in study mode

### Swipe Navigation
- Swipe left to advance to next card
- Swipe right to go back to previous card
- Works on all touch devices

### Automatic Difficulty Rating
- Response time tracking (< 3s = Easy, 3-8s = Good, > 8s = Hard)
- Auto-selected button pulses to show rating
- Auto-advances after 2 seconds
- Manual override available

### Paul Noble Theme Support
- Complete theme matching "Unlocking Japanese" book
- Chapter-based structure template
- JSON template for manual content entry

## Languages Supported
- Japanese (Hiragana, Katakana, Kanji, Numbers, Colors, Words, Sentences)
- French (Alphabet, Numbers, Colors, Words, Sentences)
- Gujarati (Vowels, Consonants, Numbers, Colors, Words, Sentences)
